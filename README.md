# @openlibrary/ui

> react-component-library

[![NPM](https://img.shields.io/npm/v/@openlibrary/ui.svg)](https://www.npmjs.com/package/@openlibrary/ui) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save @openlibrary/ui
```

## Usage

```tsx
import React, { Component } from 'react'

import {MyComponent} from '@openlibrary/ui'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
